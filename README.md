# Accounting Ledger Application Project

**Project Overview**

Flex Spend is a command-line financial management application built in Java that helps users track personal or business financial transactions. The application allows users to record deposits and payments, store transaction history in a CSV file, and generate useful ledger reports for reviewing financial activity.

**This project was designed to demonstrate Java fundamentals including:**

Object-Oriented Programming (OOP)


File I/O (FileReader, FileWriter, BufferedReader, BufferedWriter)


ArrayLists


Custom Classes


Exception Handling


User Input Validation


Date and Time Formatting



**Application Features:**


**Home Screen**
